# linux-labs

Structured Linux practice repository aligned with RHCSA and system fundamentals.

This repo documents hands-on exercises, notes, and small scripts built while practicing core Linux domains.

---

## Domains Covered

### 1. Filesystem
- Directory navigation
- File manipulation
- Permissions and ownership
- Disk usage and mounting

### 2. Users & Permissions
- User creation
- Groups
- sudo configuration
- Access auditing

### 3. Processes
- Process inspection (ps, top, htop)
- Signals and job control
- Background tasks
- Resource monitoring

### 4. Networking
- Interface inspection
- Port monitoring
- Basic troubleshooting
- Firewall fundamentals

### 5. Services
- systemctl usage
- Service troubleshooting
- Logs and journalctl

---

## Method

Each domain contains:
- Commands practiced
- Notes
- Mini scripts
- Observations from lab testing

This repository grows with consistent repetition and refinement.
